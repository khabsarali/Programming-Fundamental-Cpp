# Programming Fundamentals in C++

This repository is created for **C++ Programming Fundamentals lab tasks and projects**.  
It contains practice programs, lab exercises, and small projects completed during the course.

The code in this repository follows the Programming Fundamentals syllabus and focuses on building a strong understanding of core C++ concepts and logical problem-solving skills.

## Topics Covered
- Basics of programming  
- Variables and data types  
- Operators and expressions  
- Control structures  
- Arrays  
- Functions  
- Strings  
- Pointers  
- File handling  

## Purpose
- Lab task submission  
- Practice and learning  
- Quick reference for C++ basics
